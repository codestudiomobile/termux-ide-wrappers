# termux-ide-wrappers
Advanced compiler wrappers and terminal orchestration tools enabling lightweight IDE capabilities directly inside Termux.

---

A lightweight development toolkit providing intelligent wrapper scripts to transform a standard Termux terminal into a highly functional IDE environment. Designed to minimize compilation overhead and automate workspace mechanics.

### Features
* **C/C++ Automation (`ccr`):** Compiles via Clang/Clang++, executes binary code safely, and auto-cleans compiled footprints.
* **C# Integration (`cscr`):** Smooth compilation using `mcs` and automated runtime execution under Mono.
* **Terminal Management:** Instant environmental restarts utilizing custom intent broadcasting for integrated apps.
